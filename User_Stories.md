# User Stories

## Passenger Stories
1. As a passenger, I want to register an account so that I can book rides.
2. As a passenger, I want to book a ride so that I can travel to my destination.
3. As a passenger, I want to track my driver in real-time so that I know their location.
4. As a passenger, I want to rate drivers so that I can share feedback on my experience.
5. As a passenger, I want to cancel a ride so that I can avoid unwanted bookings.
